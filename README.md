### Description of Website:
**Lettuce Relax** The initial load page will introduce you to food consumption and exercise worldwide through statistics, focusing on the  health of all populations (debating whether or not to limit scope to youth or include everyone). The surprise is the interactive part where the visitor will be able to learn what food will provide them with what caloric intake, and what exercises they can do to burn off said calories (based on their traits?). The site will allow them to craft their own personalized dietary and workout goals, the progression of which will also be displayed with data visualization tools listed below.

### Install Guide:
1. **Clone the repository**: `git clone git@github.com:amethyst-is-cool/bacon_bros.git`
2. **Navigate into directory**: `cd bacon_bros`
3. **Create virtual environment**: `python3 -m venv venv`
4. **Activate virtual environment**:
  - maxOs/Linux: `. venv/bin/activate`
  - Windows: `venv\Scripts\activate`
5. **Install dependencies**: `pip install -r requirements.txt`

### Launch Codes:
In the terminal, access the project root directory ('cd app') and run the following commands:
- `~$ python build_db.py`
- `~$ python __init__.py`

### Launch Website
Using the provided, locally hosted link, access the website. To deactivate the virtual environment, type `deactivate`.
